3-Ques. What is Event Bubbling? And how does it work?
Ans: Event Bubbling Event Bubbling is a process where an event starts from the target element and then moves upward to its parent elements.

4-Ques. What is Event Delegation in JavaScript? Why is it useful?
Ans: Event Delegation is a system where we don't use different event lisener in every child, we use a event listener to a parent element. It's call Event delegation.

5-Ques. What is the difference between preventDefault() and stopPropagation() methods?
Ans:preventDefault() Stops the default browser behavior. stopPropagation() Stops the event from bubbling to parent elements.
